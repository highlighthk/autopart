Get Started

1. Install Environment

  1) install NVM

     https://github.com/coreybutler/nvm-windows/releases

     Download nvm with nvm-setup.exe

  2) Install Node.js version 18.19.0

     Open the cmd window in the folder of your project

     Run this command: “nvm install 18.19.0”

  3) Install nodemon

     Run this command: “npm install -g nodemon”

2. How to run automatically

  1)Locate your Bat File:

     Find the location of your bat file on your computer.

  2) Create a Shortcut:

     Right-click on your bat file.

     Choose "Create Shortcut." This will create a shortcut to your bat file in the same directory.

  3) Move the Shortcut to the Startup Folder:

     Press `Win + R` to open the Run dialog.

     Type `shell:startup` and press Enter. This will open the "Startup" folder.

     Copy the shortcut you created in step 2.

     Paste the shortcut into the "Startup" folder.
